### (2) third-party: LCM

一个 MIT Biomimetic Robotics Lab 和 MIT Locomotion Group 都在用的第三方库：Lightweight Communications and Marshalling (LCM)，是一组用于通信的库，与ROS中的话题类似，有着高带宽低延迟实时性强的特点。

该[仓库地址](http://lcm-proj.github.io/)。
